# Security policy

## Supported versions

Security fixes are applied to the latest version on the default branch.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting feature for this
repository. Do not open a public issue containing exploit details, secrets, or
personal data.

Include a concise description, affected version, reproduction steps, impact,
and any suggested mitigation. Reports will be acknowledged as soon as
practical.

## Local data

liteVDM stores `sessions.json` beside the application. It can contain full
paths to locally installed programs. The file is excluded from version control
and should not be shared.
